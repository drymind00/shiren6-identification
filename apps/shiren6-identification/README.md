# シレン6 識別補助ツール

`outputs/shiren6_notion_import/シレン6 アイテム図鑑・値段識別_インポート用.csv` を読み込む、ローカル用の単一画面ツールです。

## 起動

リポジトリのルートで次を実行します。

```powershell
.\tools\start_shiren6_identification_tool.ps1
```

ブラウザで `http://localhost:8765/apps/shiren6-identification/` が開きます。終了するには、PowerShell で `Ctrl+C` を押します。

## 使い方

1. ダンジョンを選ぶ。
2. 必要ならカテゴリ、店で確認した価格、道具の状態（通常・祝福・呪い）で候補を絞る。
3. 見つけた・確認した候補にチェックを付けると、候補一覧の下に回る。
4. 選択をやり直す時は「選択中を解除」でチェックをまとめて解除する。

絞り込み条件は残し、チェックだけをリセットします。次の周回で同じダンジョンを続ける場合に選び直す手間を減らすためです。

価格は、通常はCSVの買値・売値、祝福は通常価格の2倍、呪いは通常価格の0.87倍（小数点以下切り捨て）として扱います。杖・壺などはCSVの容量・回数ごとの価格も候補に含め、価格で絞り込んだ場合は一致した容量・回数を候補カードに表示します。「容量1ごとに買値+100・売値+40」と記録された壺も、拾得時にあり得る容量だけを候補にし、実際の容量を表示します。武器・盾は強化値-1〜+3（+1ごとに買値+100、売値+40）も候補に含め、一致した強化値を表示します。
